# 🎨 C++ Paint Application

A desktop paint application built in **C++** featuring an interactive graphical interface for drawing, editing, and managing shapes.

The application supports multiple drawing tools, shape manipulation, color customization, and undo/redo functionality through a custom GUI.

---

## 📝 Note

This project was developed using **Bobcat UI**, a course-provided GUI framework built on top of **FLTK**. Since Bobcat UI is not included in this repository, additional dependencies are required to build and run the application. The source code is provided to showcase the project's implementation and architecture.
